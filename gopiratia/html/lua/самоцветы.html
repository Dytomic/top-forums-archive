              Нужно выписать все гемы в Stoneinfo.txt "ItemHint" должны совпадать со значениями в Script.luа

              Далее нужно создать bat-файл в папке с клиентом, название любое 

              Содержимое для файла: start system\game.exe startgame table_bin

              Копию Stoneinfo.txt поместить в Клиент\scripts\table

              Запустите bat-фай. Будет много ошибок, игнорируйте их. Это из-за отсутствия других txt-файлы, не обращайте внимания. Если в Stoneinfo.txt нет ошибок, то в папке table появится новый bin-файл.
