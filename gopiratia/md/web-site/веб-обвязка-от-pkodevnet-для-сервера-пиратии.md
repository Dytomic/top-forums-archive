## Original Post
**Author:** Go-piratia  
**Date:** 2023-10-11

**Ссылка для скачивания**: [https://disk.yandex.ru/d/S6sGb0-80ywc3w](https://disk.yandex.ru/d/S6sGb0-80ywc3w)

	 

	**Особенности**

	1) Управление новостями;

	2) Страница правил сервера;

	3) Загрузки;

	4) Рейтинг игроков (по опыту, по золоту, по количеству членов гильдии);

	5) Регистрация игрового аккаунта с активацией по электронной почте;

	6) Восстановление игрового аккаунта по электронной почте;

	7) Игровая база данных монстров и предметов;

	8 ) Страница пожертвований;

	9) Статус сервера и статистика, GM online;

	10) Управление учетной записью (информация об учетной записи, символы, смена пароля/ электронной почты/ pin-кода);

	11) Автономное расширение сервера киосков для просмотра киосков и поиска товаров.

	**Требования**

	1) Веб-сервер;

	2) PHP версии 7.0 и выше;

	3) Включено расширение PDO с драйверами sqlsrv и sqlite.

	**Установка**

	1) Поместите файлы веб-сайта в рабочий каталог вашего веб-сервера;

	2) Конфигурационный файл config\inc.config.php . Файл очень хорошо прокомментирован, так что у вас не должно возникнуть никаких проблем с его настройкой;

	3) Сайт использует движок шаблонов Smarty для легкого изменения внешнего вида. Все HTML-шаблоны хранятся в папке templates;

	4) Вы можете найти шаблоны электронной почты в папке templates\email.
