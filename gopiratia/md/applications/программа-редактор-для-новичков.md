### Reply #1
**Graf** — 2023-12-25

Changelog от 25.12.2023:

	1. В списке файлов теперь не отображаются файлы, с расширением .bin;

	2. Обновил список поддерживаемых txt файлов (если нужны другие, пишите.. Добавлю):

	![image](https://pkodev.net/uploads/monthly_2023_12/image.png.6679182572591b47b65babeb6c28aa40.png)

	3. Поиск файлов теперь не затрагивает регистр;

	4. Изменения в файл теперь вносятся без нажатия "Изменить". Автоматически после внесения изменений в таблицу;

	5. После изменения файла, будет появляться информация о времени изменения:

	![image](https://pkodev.net/uploads/monthly_2023_12/image.png.3b535ec61f21e6dde52742307365bfa2.png)

	P.S. лично я постоянно открываю файл для проверки, так спокойнее)

	6. Добавил мультиязычность.

	В каталоге /lang хранятся языки. Вы можете добавить любой язык в каталог, заменить строки и у вас будет свой язык.

	Ссылки обновил.

### Reply #3
**Graf** — 2023-12-26

Changelog от 26.12.2023:

	1. Добавлена оптимизация под разный размер окна приложения. То есть теперь можно расширять окно или открыть "полноэкранный режим";

	2. Добавлено горизонтально меню;

	3. Добавлена кнопка "Выбор каталога". Теперь программа может использовать файлы не только, находящиеся рядом с ней;

	4. Добавлена мультиязычность для горизонтального меню;

	5. Обновлен список строк в языковых файлах;

	6. Добавлена очистка строки выбора файла при изменении директории, чтобы избежать ошибок;

	7. Добавил ассинхронное чтение файлов, чтобы ускорить вывод информации в таблицу;

	8. Добавлена кнопка "Выход", чтобы закрыть приложение;

	9. Добавлена кнопка "Список обновлений", там показывается последний ChangeLog;

	10. Добавлена кнопка "О программе";

	11. Добавлена кнопка "О авторе";

	12. Добавлен коопирайт в нижнем правом углу.
