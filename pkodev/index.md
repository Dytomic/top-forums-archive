# PKO DEV — Site Overview

**URL:** https://pkodev.com  
**Type:** Community development forum  
**Focus:** Tales of Pirates (ToP) and Pirate King Online (PKO) private server development

PKO DEV is a developer-centric forum for the Tales of Pirates / Pirate King Online game engine. It serves as the central hub for source code releases, server setup guides, Lua scripting, client modding, and community collaboration around private server development.
