To make the most out of your thread for Server Advertisement for your server, make sure you post the following:

- Server Name

- Server Launch Date

- Website link and/or discord link

- What type of server it is, if its PK, Vanilla or something else

- How long its been up for if its not new

- Images of the server, with its logo and/or screenshots/videos of the server if you have any.

All this info helps people look for intressting new servers that they can check out!
