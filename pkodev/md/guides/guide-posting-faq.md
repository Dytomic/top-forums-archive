## Original Post
**Author:** Zakernado

To make an efficient post here in Guides, ensure that you follow the criteria below:

- Post as detailed of a guide as you can, there may/may not be non native english speakers around, so keep that in mind

- If the guide is not from you, make sure to credit the original author.

- Make sure to use the correct prefix for the guide

- Do not post any guide non related to Top/PKO
