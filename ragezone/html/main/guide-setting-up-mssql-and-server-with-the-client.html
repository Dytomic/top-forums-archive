ok, i have installed VMWare, went through a full installation of MSSQL 2000, restored the DB, setup accounts within MSSQL 2000, edited the .cfg files and got a working server. i have taken screenshots step by step. All files (excluding MSSQL 2000 which the link is included in the PDF file) are included in the DBBackup.exe file, they are the DB Backup file, Log/Data file, PDF How-TO and .cfg files for the 1.33 chinese server. it will place them in the correct directory of C:\DBBackup\ to make using the tutorial simpler. After you have ran the installer, goto C:\DBBackup\ and open the PDF file in there to start the walkthrough. I really hope this helps everyone in setting up MSSQL 2000 and hope it cuts down on people asking the same question over and over.
