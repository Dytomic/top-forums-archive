Client -
1. Get and install the Taiwanese 1.33 client from here:

To view the content, you need to sign in or register


Please don't use another versions/languages, because you won't be able to install our patch on it.
If someone is experiencing difficulties getting it, please reply here, so someone could reupload it.

2. Install this patch. (Run and then point it to a folder where you installed your TW 1.33)
To view the content, you need to sign in or register

If you get an error when running this patch, try this archive with patched files
To view the content, you need to sign in or register


3. Use the program from
To view the content, you need to sign in or register
to edit the IP address of your server. Also you can edit it manually. The IP addresses are in \scripts\table\ServerSet.bin file. Just change 127.0.0.1 with the IP of your server (with trailing 0x00) using the hex editor of your choice.

4. Your client should turn into English now
Remember to run it using "start.bat" file in the root game directory, the official launcher/updater is removed by our patch.
