Depends on your SQL Server, and how you set it up. This requires you to run the "SQL Browser" Process.

You can turn it on via the SQL Configuration, or going to

Control panel -> Administrative Tools -> Processes. Look for the "SQL Browser" Process.
press Properties, and put it to auto. Then close properties and look @ the left panel. Press start.

Check on the options box to make sure your sql settings are correct.

The password is the one you entered into the sql database. usually for these servers it's
"Y87dc#$98".

They've encrypted the password on the Server files because they had to make sure h4xers weren't able to enter the DB directly.
