General

The first column is the id you will give the item.
The second colum is the name of the item.
The third column is the icon tga name wich are in your cliente folder \texture\icon.
The fourht column is the model the script or something like that used for mount the images on to the char.
The fifht column is the texture for lance \texture\character
The six column is the texture for carsise \texture\character
The seven column is the texture for phyllis \texture\character
The eight column is the texture for ami \texture\character
The other columns not a clue.
The last column is the description column.

ERROR

-If when you do &make the game crashes saying someting about id bla bla bla, and when you try to log ing again same error, that cause the ItemInfo in the server is not the same as the one in the client.
So copy the iteminfo.bin of the server to the script\table in the cliente.

-If after "succesful" added a item if dont show well like some party of the body dissapears, that cuase there is no texture. So check again that for that item it exisxt in the cliente folder the texture you have put.

-If after "succesful" added a item if dont show well like boots donsnt suit the char, the texture you put instead of Lance is the one of phyllis/carsise/ami so check again that that texture donest suit another char.
