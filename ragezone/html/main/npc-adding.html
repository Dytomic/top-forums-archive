If you want add NPC in :
1.Go to Server files/resource/garner
2.Open garnerNPC.txt
3.Copy and paste this Line to ur garnerNPC.txt
163 (NPC NAME) 1 979 0 23000,274800 230800,27400 270 Argent City 20 0 trade_09 0
Click to expand...

4.Edit the Coords
230800,274800 230800,274800
Click to expand...
Make like 222200,222200 and same again

Every time you have to do the 00 After the coords!

Now it will look like this:
163 Npc Name 1 979 0 230800,274800 230800,274800 270 Argent City 20 0 trade_45 0
Click to expand...

Npc is Done!
