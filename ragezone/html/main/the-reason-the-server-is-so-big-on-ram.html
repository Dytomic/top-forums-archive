Hello all!

I know you are kind of grouchy when you found out it takes 3 GBs to run the ToP Server on your ram, but there are a few reasons for that.

1: ToP Is OLD.
2: Anti-Terrain hax & anti-haxing system.
3: Persistence is apparently the answer from the original creators of ToP.

The ToP Server was Top of the line technology (For them) at the time.

(To tell you the truth, this game uses so much ram/cpu that it really eats your computer up with it's awful graphics. They don't even have a "True" atmosphere.)

Since nowadays, even the most recent servers still "Load" the maps, it really isn't needed.

First off, Loading a file onto your ram via a program/server will only increase the amount of space you need to run the server.

The best way to prevent this huge amount of ram needed in a server, is just read the info from the file line-by-line via request. Although this will remove the anti-terrain hacking ability(To some extent), it will greatly improve server speed and client connections.

The best way to read line-by-line is to index the IDs needed on each line for later use.
