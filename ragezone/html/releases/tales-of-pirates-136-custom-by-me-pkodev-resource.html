Okay I will be sharing my files and this is made based on a Solo Server adventure. There are few resource yet to be implement such as Azz gem but it existed in the files.

Feature -
1. Contract petal ( done inside DW, DS and FC, each contract reward 1 CP and 1M ) - exchange for Nightmare helm, Log Pose (for boss summons), Sandbag lv5
2. Accessories fragment from lone tower
3. Nightmare helm upgrade from CA mobs
4. Kal exchanger for CA set.
5. Fairy mall for marriage.
6. Great gem chest from winter 100+ mobs.
7. Hell maps for kylin fragment.
8. Kylin set if upgraded will have extra effect. the item for upgrade is not yet implement but the system is in place. Need to put a system to get it inside the game.
9. Mobs from DS DW and FC rewards Mall Point tears that can be used to buy from in game mall.
10. Reforge card available.
11. Special custom - buff from buffer class increase a lot of stats to the targeted character based on its highest stats. Eg, if STR is the highest, the value of STR will be a multiplier for the other stats increase. Same goes across the board.
12. Ref is cheap and unlimited, other gem need to be farm from lv1 to lv4. Lv5 to lv9 can be exchanged in npc for broken, cracked and chipped. - contract petals.
13. Gem combiner for all gem
14. custom gems from Thundoria Mirage.
