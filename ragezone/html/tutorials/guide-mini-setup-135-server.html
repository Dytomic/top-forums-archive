[Guide] Mini Setup 1.35 Server (Sql Server Updated)

  1. First Download 1.35 Server Files :
    To view the content, you need to sign in or register
  2. To view the content, you need to sign in or register
  3. Extract Server Files
  4. Install
    To view the content, you need to sign in or register

  1. Open Enterprise Manager >> Ms. Sql Server >> Sql server Group >> Local >>Database
  2. Right Click on Database >> All Task >> Attach Database (Goto Database in The 1.35 Server) >> OK (Another 1 Too )
  3. Now Goto Security >> Login >> Right Click >> Creat New Login >> Name it game (for GameDB) or account (for AccountServer)
  4. Use Sql Server Authentic >> type There Y87dc#$98 >> Database >> Change to GameDB (game) or AccountServer (account)
  5. next on Server Roles Tab >> Click on System Administrator
  6. and Last goto Databse Acces Tab >> CLick on GameDB (game) or AccountServer (account) >> also Click db_owner >> Ok And type The Password Y87dc#$98 too.

When make login and Got "Already exist" , its Normally, Because the Database Already Have it.

And Now try to Open Your AccountServer, GateServer, GroupServer, and GameServer.

Congratulation Now Your Server Running.
